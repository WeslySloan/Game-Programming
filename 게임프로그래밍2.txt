지속적으로 학습하여 스스로 성장하는 기술 / 게임 어뷰징 검출 사례 등

2. 렌더링에 대해 기술하시오.
   (DOM, CSSOM, Render tree 포함)
퓨터 프로그램을 사용하여 씬 파일 로 부터 영상을 만들어내는 과정

3. Repaint, Reflow에 대해 기술하시오.
Repaint - 화면에 변화가 있을 때 화면을 그리는 과정
Reflow - 렌더링 엔진에서 요소를 배치하는 과정

4. 아래의 클래스에 대해 간단히 기술하시오.
   MonoBehaviour, Transform, Rigidbody
MonoBehaviour - 모든 unity 스크립트가 파생되는 기본 클래스
Transform - 스크립트를 통한 게임 오브젝트의 계층적 관계에 대한 다양한 작업 방법을 제공
Rigidbody - 게임 오브젝트가 물리 제어로 동작하게 함

5. 아래 함수의 기능에 대해 간단히 기술하시오.
   Start, Update, Awake, OnGUI, OnMouseOver, OnCollisionEnter, OnTriggerEnter
Start - 1번만 실행되는 함수로 Update 함수보다 우선적으로 실행됨
Update - 매 프레임마다 호출되는 함수
Awake - 게임을 시작하기 전에 변수 등을 초기화하기 위해 사용
OnGUI - GUI이벤트를 렌더링하고 처리하기위해 사용
OnMouseOver - 마우스가 요소에 있는동안 호출
OnCollisionEnter - Collision에 들어갔을때 호출
OnTriggerEnter - trigger 안에 들어갔을때 호출
